# 📧 Bulk Mail Application (MERN Stack)

A full-stack **Bulk Mail Application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.
This app allows users to send emails to multiple recipients efficiently with a simple and intuitive interface.

---

## 🚀 Features

* Send bulk emails to multiple recipients
* User-friendly dashboard for composing emails
* Dynamic email form handling
* Backend API for sending emails
* Success & error handling for mail delivery
* Scalable and modular architecture

---

## 🛠️ Tech Stack

### Frontend

* **React.js** – UI development
* **CSS / Tailwind CSS** – Styling
* **Axios** – API calls

### Backend

* **Node.js** – Runtime environment
* **Express.js** – Backend framework

### Database

* **MongoDB** – Data storage
* **Mongoose** – ODM for MongoDB

### Other Tools

* **Nodemailer** – Sending emails
* **dotenv** – Environment variables
